Création du jeu Bang en java

Nous avons lors d'un TP noté du réaliser un jeu Bang en Java en partant d'un code avec quelques fonctions.

Pour cela nous avons été par groupe de 3 ou 4.

Nous avons du implémenter toutes les cartes correspondant aux personnages ainsi que les cartes à jouer. Nous avons aussi du implémenter de nombreuses fonctions du jeu.
